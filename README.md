## THIS IS THE TERM PROJECT

TO run the project locally follow the below steps:

conda create -n myenv python=3.6.12
conda activate myenv 
python app.py 

Pre-Requsities: 

To run the project successfully, please make sure that you have installed anaconda and python in your system.
